# 👨‍💼 Employee Management System

A Python-based Employee Management System designed to manage employee records efficiently.

## ✨ Features

- Add Employee
- View Employee Records
- Search Employee
- Update Employee Information
- Delete Employee Records
- Store Data in a Text File

## 🛠 Technologies Used

- Python
- File Handling
- CRUD Operations

## 📂 Project Structure

Employee-Management-System/
│── main.py
│── employees.txt
│── requirements.txt
│── README.md

## ▶️ How to Run

1. Install Python.
2. Clone or download this repository.
3. Run:

```bash
python main.py
```
